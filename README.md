# MLB Live Scoreboard

A self-contained, auto-refreshing scoreboard of all of the day's MLB games. It
polls the keyless [MLB StatsAPI](https://statsapi.mlb.com) schedule endpoint
directly from the browser every ~25 seconds and updates in place — no server,
cron, or build step. Games are grouped into Live / Scheduled / Final / Postponed,
with score, inning state, bases, count, outs, and probable pitchers.

## Run locally

```bash
python -m http.server 8000     # then open http://localhost:8000
```

## Deploy

Hosted on GitHub Pages from this repo's `main` branch (root). Any static host
works too (Netlify, Cloudflare Pages, S3) — just serve these files.
